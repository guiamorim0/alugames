# AluGames 🎮

Este projeto foi desenvolvido durante um curso da **Alura**, com o objetivo de praticar **lógica de programação e JavaScript** através da construção de uma aplicação simples.

## 📚 Sobre o projeto

O **AluGames** é um projeto criado para exercitar conceitos fundamentais de programação, como:

* Manipulação do **DOM**
* Uso de **JavaScript**
* Estrutura básica de **HTML e CSS**
* Lógica de programação aplicada a interações da página

## 🚀 Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## 🎯 Objetivo

O principal objetivo deste projeto é colocar em prática os conhecimentos aprendidos durante o curso da **Alura**, desenvolvendo um pequeno sistema funcional e reforçando conceitos importantes da programação.

## 📖 Curso

Projeto desenvolvido como parte dos estudos na plataforma **Alura**.
